## 说明

* 不区分大小写
* 按上下键可以查看历史命令
* tab键补全
* 使用/?或/help可以查看命令用法，用法中的[]代表可选

## 基本命令

| 命令  | 功能 |
| ----- | ---- |
| echo  | 显示 |
| pause | 暂停 |
| cls   | 清屏 |
| exit  | 退出 |

## 常用命令

| 命令               | 功能             |
| ------------------ | ---------------- |
| type               | 显示文件内容     |
| find               | 查找字符串       |
| fc                 | 比较文件内容     |
| data               | 显示修改日期     |
| time               | 显示修改时间     |
| ping               | 网络连接测试     |
| ipconfig           | TCP/IP信息       |
| set/a              | 计算表达式       |
| shutdown           | 关机             |
| ctrl + pause break | 中断执行中的命令 |

## 文件操作命令

路径分隔符：`\`

当前目录`.`

上一级目录：`..`

| 命令        | 功能               |
| ----------- | ------------------ |
| dir         | 显示文件内容       |
| tree        | 显示文件夹树形结构 |
| 盘符：      | 进入指定磁盘       |
| cd          | 切换目录           |
| mkdir或md   | 创建目录           |
| rmdir或rd   | 删除目录           |
| del         | 删除文件           |
| rename或ren | 重命名文件         |
| copy        | 复制文件           |
| move        | 移动文件/重命名    |
